# Sanitized benchmark inputs

Benchmark instances are synthetic public problem-class instances of
Equation (1) in the paper (budget, box, exposure-band, turnover, and policy
constraints over an N-asset universe). Instance generators record N, seed,
constraint bundle, and tolerance gates; no client data and no proprietary
engine state are present in any input.

`benchmark_inputs_manifest.csv` lists instance families used by each evidence
artifact.
